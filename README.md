# Customer Churn Early Warning Project

Bu projede bir bankanın müşteri verileri üzerinde **müşteri terk (churn) tahmini** yapan
bir **erken uyarı modeli** geliştirmeyi amaçlanmaktadır.

## Veri Seti

Proje kapsamında kullanılan veri seti, Kaggle üzerinde paylaşılan aşağıdaki kaynaktan alınmıştır:

- https://www.kaggle.com/datasets/radheshyamkollipara/bank-customer-churn

## Kullanılan Teknolojiler

- Python 3.10+
- NumPy
- pandas
- scikit-learn
- Matplotlib

## Kurulum

```bash
pip install --upgrade pip
pip install numpy pandas scikit-learn matplotlib
python main.py
```
